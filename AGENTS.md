# TEMTECH Dashboard CRM - AGENTS.md

## Roles

### Agent 1: Project Manager

Responsable de:

* Planificación
* Proyectos
* Tareas
* Timelines
* Presupuestos
* Comunicación con cliente

### Agent 2: Growth Hacker

Responsable de:

* Leads
* Embudo de ventas
* Conversión
* Remarketing
* Estadísticas de crecimiento

### Agent 3: Full Stack Developer

Responsable de:

* Código Next.js + TypeScript
* Supabase
* Integraciones
* Despliegue
* Escalabilidad

### Agent 4: UI/UX Designer

Responsable de:

* Diseño visual
* Experiencia de usuario
* Animaciones
* Prototipos interactivos
* UI moderna y premium

### Agent 5: QA & Security Tester

Responsable de:

* Testing
* Seguridad
* RLS
* Performance
* Bugs

---

## Flujo de Trabajo

1. Project Manager define especificaciones.
2. UI/UX Designer crea diseño.
3. Full Stack Developer construye.
4. Growth Hacker define métricas.
5. QA Tester valida todo.

Todos los agentes trabajan simultáneamente pero coordinados.
